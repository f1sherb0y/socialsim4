# 快速上手

欢迎使用“社会科学实验装置”！本平台旨在为您提供一个强大而灵活的仿真环境，通过构建和观察智能体（Agent）在特定社会情境下的互动，探索复杂的社会现象。

本手册将详细引导您完成从零开始到运行一个完整仿真实验的每一步。整个过程可以概括为：**选择模板 → 设计方案 → 配置智能体 → 配置AI模型 → 确认并运行**。

## 一、进入登录与注册界面

**登录操作**: 

1. 在“用户名”输入框中填写您的注册用户名。
2. 在“密码”输入框中填写您的账户密码。
3. 点击蓝色的 **登录** 按钮即可进入系统。

 <img src={img1} alt="alt text" />


**首次使用？** 请先注册: 如果您还没有账户，请点击登录框下方的“还没有账户？立即注册”链接，跳转到注册页面。 

1. **用户名**: 输入一个您选择的用户名。
2. **电子邮件**: 输入您的有效电子邮箱地址。
3. **姓名**: 输入您的真实姓名。
4. **密码**: 设置您的账户密码。
5. **确认密码**: 再次输入您设置的密码以进行确认。
6. 点击蓝色的 **注册** 按钮，完成账户创建。


 <img src={img2} alt="alt text" />

## 二、步骤一：选择仿真模板

登录后，您将开启实验设计的第一步。这里的“公共模板”是为您精心预设的实验起点。**每一个模板都包含了一套完整的默认配置**，包括：

- **预设的智能体**: 具有不同背景、性格和数量的虚拟角色。
- **预设的事件**: 围绕特定主题（如公共政策讨论）设计的系列情境。
- **预设的工作流**: 智能体进行思考和行动的基础逻辑框架。

选择一个与您研究方向最接近的模板，可以大大简化后续的配置工作。点击模板下方的 **选择** 按钮，以该模板为基础，开始您的个性化设计，例如：

 <img src={img3} alt="alt text" />


1. 浏览“公共模板”部分提供的不同仿真场景，例如“生命讨论立法会”或“公共事件讨论委员会”。
2. 点击您想要进行的实验模板下方的 选择 按钮，进入下一步方案设计。


## 三、步骤二：设计仿真方案

这是实验设计的核心环节。您将在这里定义实验的具体内容和运行逻辑。此界面分为三个关键部分：
1. **基本信息**:
- **实验名称**: 为您的本次实验取一个清晰、有辨识度的名称。
- **初次仿真轮数**: 设定实验开始后，系统自动连续运行的仿真轮次。例如，设置为“5”，则开始后会自动进行5轮智能体间的互动。
2. **事件列表与详情** (定义社会情境): 在这里，您可以构建仿真的故事情节。
- **添加/编辑事件**: 每个事件都代表一个特定的社会情境或讨论议题。您可以点击“+添加事件”来创建新事件，或点击已有事件进行编辑。
- **配置事件细节**: 对于每个事件，您都可以定义：
  - **事件详情**: 对该情境进行详细、丰富的描述。
  - **相关政策**: 提供与事件相关的背景资料或规则。
  - **补充搜索内容**: 输入关键词，智能体在决策时会利用这些补充的搜索内容。
3. **流程设计** (为智能体注入思考模式 - 高级功能): 此部分决定了智能体如何“思考”。每个智能体都会遵循“计划”和“执行”的顺序来调用大语言模型（LLM）。我们开放了对底层Prompt（提示）的编辑权限，让您可以精细地调整智能体的行为逻辑。
- 编辑Prompt: 点击文本框，您可以修改：
  - **第一行-任务具体指令**: 告诉智能体具体要做什么。
  - **第二行-期望推理格式**: 引导智能体如何思考和分析问题。
  - **第三行-期望输出内容**: 规定智能体最终回应的内容。
完成所有设计后，点击 **下一步。**

## 四、步骤三：自定义智能体

在这一步，您将配置仿真世界中的“演员”——智能体。您可以基于模板的预设进行修改，也可以创建全新的智能体。
- **手动配置**:
  1. 点击蓝色的 **“+添加智能体”** 按钮。
  2. 在右侧详细填写智能体的各项属性，以塑造一个丰满、独特的角色，包括：
    - 姓名
    - 年龄
    - 生活作息
    - 个性特征
    - 背景故事
- 从文件快速生成:
  1. 点击绿色的 **“根据描述生成”** 按钮。
  2. 上传一个包含智能体描述的文本文件。
  3. 系统将自动读取文件内容，并为您生成智能体的各项属性，大大提高配置效率。
完成所有智能体的配置后，点击 **下一步。**

<img src={img5} alt="alt text" />

## 五、步骤四：配置仿真参数

此步骤是将您的仿真实验与强大的大语言模型（LLM）“大脑”连接起来的关键。智能体的所有对话、思考和行动都将由您在这里配置的模型生成。
1. **API 服务地址 (Base URL)**: 输入您的AI服务提供商的API地址 (例如 OpenAI, Google AI 等)。
2. **API 密钥 (API Key)**: 输入您账户的有效API密钥，以获得访问权限。
3. **模型名称 (Engine/Model)**: 从下拉菜单中选择或输入您希望使用的具体模型（如 gpt-4, claude-3-opus 等）。
4. **高级参数 (可选)**: 您可以拖动滑块微调模型的表现，如“模型创造力 (Temperature)”或“回复最大长度 (Max Tokens)”。
确认所有信息无误后，点击 下一步。


<img src={img6} alt="alt text" />

## 六、步骤五：确认方案并开始仿真

这是启动实验前的最后一步，请仔细核对您的所有配置：
- **模板信息**: 确认实验的基础框架。
- **智能体配置**: 检查角色数量和核心设定。
- **方案设计**: 确认事件数量和内容。
- **模型参数配置**: 再次检查AI模型的设置。
如果一切准备就绪，点击中央的 开始仿真 按钮，您的社会科学实验就将正式启动！如果需要调整，随时可以点击 < 返回上一步。

## 七、监控仿真运行

进入此页面，代表您的仿真世界已经开始运行。
- **实时观察**: 在中央的对话窗口，您可以实时看到各个智能体根据您的设计所进行的对话和互动。
- **动态干预**: 仿真过程中，您不是一个纯粹的旁观者。您可以使用“发布事件”等功能，向仿真世界中动态引入新的变量，观察智能体如何应对。
- **耐心记录**: 请耐心观察实验进程，并记录对您有价值的数据和现象。
