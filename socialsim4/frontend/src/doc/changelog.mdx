# 更新记录

## Version 2.0 Release Notes

**发布日期：2025-06-16**

本次 **2.0 版本** 聚焦于 **智能交互效率**、**系统灵活性** 与 **可视化体验** 的全面提升，增强了用户与多智能体系统的交互便捷性与可控性，具体更新内容如下：

---

### 新增功能

#### 一键总结功能

在仿真结果展示界面新增“一键总结”按钮，支持快速生成当前仿真内容的摘要，提升任务后分析效率。

#### 智能体配置文件解析

系统新增支持从 `.txt` 等格式文档中读取配置内容，自动生成智能体初始设定，方便用户自定义部署多个智能体。

#### 情绪展示模块

引入基本情绪识别模型，系统可在推理过程中识别并展示智能体的情绪状态，以可视化方式增强交互拟人性。


#### 用户大模型接口配置持久化

系统现已支持每个用户**自定义配置并持久化多个大模型 API 接口**（如对话模型、生成模型、嵌入模型等）。  
用户可一次性配置对应接口信息，平台将自动保存并在每次仿真运行时**自动填充调用**，无需重复输入，大幅提升使用体验。

#### 提供详细操作文档

上线了完整的**用户操作说明文档**，涵盖仿真系统使用流程，方便用户快速上手与排查问题。
